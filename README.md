# LocalStack

LocalStack testing with terraform

## Dependencies

- Python v3.12.6
- Terraform v1.9.5
- Docker 27.2.0

## Running locally

Check the command for you OS.

1. Create Virtual Environment:

- Windows

```powershell
.\virtualenv.ps1
```

- Linux

```sh
bash virtualenv.sh
```

2. Run terraform:

- Windows

```powershell
cd .\scripts\
.\run-infra.ps1
```

- Linux

```sh
cd scripts/
bash run-infra.sh
```
